
# Elecal

## 简介

> **Elecal** 是一个迷你的 USB 电源功率计，在硬件设计上可以同时测量电源电压电流以及功率，
同时通过软件方案实现 mAh，Ah，Wh 的实时监测。
通过 mAh，Ah，Wh 这些计量单位可以很方便的计量和计算被监测电源所输出的电能。 比如其中的 mAh 所能
说明的是一颗 3000mAh 的电池可以提供在 300mA 恒流输出状态下持续 10 小时的电能输出能力。
